pub fn tx_maturity() -> Option<u64>
Expand description

Get the maturity for the transaction, if it is set.

Returns

  • [Option] - The maturity for the transaction.

Examples

use std::tx::tx_maturity;

fn foo() {
    let maturity = tx_maturity();
    log(maturity);
}
