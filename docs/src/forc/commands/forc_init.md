
# forc-init
Create a new Forc project


## USAGE:
forc init [OPTIONS] <PROJECT_NAME>


## ARGS:

<_PROJECT_NAME_>

   The name of your project


## OPTIONS:

`-h`, `--help`, 

Print help information

`-t`, `--template`, <_TEMPLATE_>

Initialize a new project from a template.

Example Templates:
- counter
