The Sway Programming Language

Parse

Usage: forc parse-bytecode [OPTIONS] <FILE_PATH>

Arguments:

<FILE_PATH>

Options:

-v, --verbose...

Use verbose output

-s, --silent

Silence all output

-L, --log-level <LOG_LEVEL>

Set the log level

-h, --help

Print help

-V, --version

Print version

EXAMPLES:

Parse bytecode

forc parse-bytecode

EXAMPLE

We can try this command with the initial project created using forc init, with the counter template:

forc new --template counter counter
cd counter
forc build -o obj

counter$ forc parse-bytecode obj
